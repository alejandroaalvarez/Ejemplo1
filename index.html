Aprende a programar con python

¿Como programar en python?

Para aprender a programar en python es importante saber algunos conceptos tales como:

VARIABLES

Estas son las que se encargan de almacenar informacion, las cuales son:

  • Cadenas de texto
  • Numeros enteros
  • Numeros decimales
  • Booleano

PRINT

Es la encargada de imprimir informacion de la cual se desea mostrar al usuario, se utiliza con los () para mostrar un mensaje o informacion

INPUT

Es la que solicita datos al usuario, para poder llenar algun tipo de información

CONVERTIDORES

Permite convertir un tipo de dato en un tipo de variable la cual denecitemos, por ejemplo:

  • Int-Número
  • Str-Texto
  • Float-Números decimales

FORMAT

Nos permite incluir en una cadena de texto, el cual tiene variedad de funciones y es utilizado dentro de un print fuera de las comillas.

LOWER

Es el que convierte el texto de mayúscula a minúscula

UPPER

Es el que convierte el texto de minúscula a mayúscula

CONDICIONAL IF

Es la encargada de generar condiciones, dentro de las condiciones se tienen dos posibles opciones o mas, en ella se pueden crear opciones predeterminadas u opciones de rechazo.
